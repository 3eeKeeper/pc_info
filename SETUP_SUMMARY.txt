✨ FEATURES
=============================================================================

HARDWARE REPORT INCLUDES:
├─ CPU: Model, cores, frequency, cache, real-time usage
├─ RAM: Total capacity, each module's specs (speed, type, size)
├─ Storage: All drives, partitions, capacity, usage
├─ GPU: Graphics cards, VRAM, drivers, resolution
├─ Network: Adapters, IPs, MAC addresses, speeds
├─ Motherboard: Make, model, BIOS version
└─ Battery: Charge level, time remaining (laptops)

SOFTWARE REPORT INCLUDES:
├─ OS: Windows version, build, installation date
├─ Programs: All installed software (200+ items typical)
├─ Startup: Programs that auto-start
├─ Services: All Windows services and their status
├─ Processes: Running programs with CPU/RAM usage
├─ Updates: All installed Windows updates
├─ Users: All user accounts on the system
└─ Environment: System configuration variables
